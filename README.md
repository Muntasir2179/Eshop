
# E-Shop (Online Super Market)

This is an web application which anables us to do shopping online. All kinds of products can be represented to sell in this platform. The application has a basic authentication system that keeps the record of the users. By the way, users are also welcome to visit the website without logging in. But if they whish to buy something they must have to login. The application keeps trac of all the orders that has been placed so far. Users can checkout by putting their address and phone number. There are no payment method added in this application.
